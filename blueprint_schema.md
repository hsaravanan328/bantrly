## Bantrly — Lesson Blueprint: One-Page Summary

---

### Every Lesson Has

| Section | Fields |
|---|---|
| **META** | `id` · `type` · `domain` · `bucket` · `duration` · `skill` · `signals` · `xp` |
| **QUEST** | `unit` → `quest` → `position` (1 / 2 / 3) |
| **ROUNDS** | 1–3 rounds, each with `content` + `model` + `practice` + `perform` |
| **REWARD** | `xp` · `badge` (position 3 only) · `message` · `unlocks` |

---

### Every Round Has 3 Steps

| Step | What Happens |
|---|---|
| **MODEL** | App demonstrates · replay allowed |
| **PRACTICE** | Student imitates · **NOT scored** · hints visible · can skip |
| **PERFORM** | Student records · **SCORED** · hints hidden |

---

scrambled_story    pace_difference    emotion_difference    character_read

### 4 Grade Buckets

| Bucket | Grades | Ages |
|---|---|---|
| 🌱 **Spark** | K–2 | 5–8 |
| 🔨 **Builder** | 3–5 | 8–11 |
| 🔭 **Explorer** | 6–8 | 11–14 |
| 🏆 **Achiever** | 9–12 | 14–18 |

---

### 2 Unbreakable Rules
```
1.  practice.scored = false   —   always, no exceptions
2.  perform.scored  = true    —   always, no exceptions
```
